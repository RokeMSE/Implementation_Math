class matrix:
    def __init__(self, matrix):
        self.matrix = matrix
    
    # Vit li Gauss Jordan Inverse do  W3 c p kiu gy li sai s
    def gauss_jordan_inverse(self):
        n = len(self.matrix)
        # To ma trn n v
        inverse = [[0 for i in range(n)] for j in range(n)]
        for i in range(n):
            inverse[i][i] = 1

        # Chuyn ma trn v dng tam gic
        for i in range(n):
            # Tm hng c phn t ln nht  ct i
            max_row = i
            for j in range(i + 1, n):
                if abs(self.matrix[j][i]) > abs(self.matrix[max_row][i]):
                    max_row = j
            
            # Kim tra xem phn t ln nht c bng 0 khng
            if abs(self.matrix[max_row][i]) < 1e-10: # D tr cho floating point error
                return None
            
            # i hng i v hng max_row
            self.matrix[i], self.matrix[max_row] = self.matrix[max_row], self.matrix[i]
            inverse[i], inverse[max_row] = inverse[max_row], inverse[i]

            # Chuyn phn t  ct i ca cc hng khc v 0
            for j in range(n):
                if i != j: 
                    ratio = self.matrix[j][i] / self.matrix[i][i]
                    for k in range(n):
                        self.matrix[j][k] -= ratio * self.matrix[i][k] 
                        inverse[j][k] -= ratio * inverse[i][k]

        # Chuyn cc hng v 1
        for i in range(n):
            ratio = self.matrix[i][i]
            for j in range(n):
                self.matrix[i][j] /= ratio
                inverse[i][j] /= ratio
        return inverse
    
    def transpose(self):
        # Chuyn v ma trn
        return [[self.matrix[j][i] for j in range(len(self.matrix))] for i in range(len(self.matrix[0]))]
        
    def print_matrix(self):
        # In li ma trn  thot format
        for i in range(len(self.matrix)):
            for j in range(len(self.matrix[0])):
                # Kim tra c phi l ch khng
                if type(self.matrix[i][j]) == str:
                    for k in range(len(self.matrix)):
                        for l in range(len(self.matrix[0])):
                            print(self.matrix[k][l], end = '')
                    return
                if j == 0:
                    print("[", end = '')
                print(self.matrix[i][j], end = '')
                if j == len(self.matrix[0]) - 1:
                    print("]")
                else:
                    print(", ", end = '')

== Class LinearRegressionModel ==¶

Mô hình hồi quy tuyến tính:¶

$$\hat{y} = _0 + _1*x_1 +...+ _n*x_n$$
  • p dng vo d liu thu c, ta c:

DIQ010 = $x_0 + x_1 *$ RIDAGEYR $+ x_2 *$ RIAGENDR $+ x_3 *$ PAQ605 $+ x_4 *$ BMXBMI $+ x_5 *$ LBXGLU $+ x_6 *$ LBXGLT $+ x_7 *$ LBXIN $

Mục đích: Xây dựng một mô hình hồi quy tuyến tính, bao gồm huấn luyện mô hình (fit), dự đoán (predict), tính toán thông số đánh giá MSE, $R^2$, và lựa chọn đặc trưng tốt nhất bằng Forward Selection.¶

Chức năng các hàm:¶

  • __init__: Khi to mt Object m hnh hi quy tuyn tnh (tng trng cho 1 model). Object gi 2 gi tr sau:

    • self.coef_: Danh sch cc h s hi quy ca cc feature.
    • self.intercept_: H s t do (bias) ca m hnh.

  • fit(X,y): Hun luyn m hnh bng cch tm cc h s hi quy v h s t do thng qua cng thc: $$\hat{x} = (X^{T}X)^{-1}X^{T}y$$

    • Cc bc thc hin:
      • Trch ma trn $X$ (ch khng c ct DIQ010) v vector $y$ (ch ct DIQ010) t dataframe gc.
      • Thm ct intercept vo $X$ : l bias vi gi tr tt c l 1 (h s t do - hay $x_0$)
      • Chuyn i $X$ v $y$ thnh dng ma trn (list Object matrix s dng cc hm bn trn).
      • Tnh ma trn chuyn v ca $X$ l $X^{T}$.
      • Tnh cc ma trn $X^{T}X$, $X^{T}y$.
      • S dng phng php Gauss-Jordan tnh ma trn nghch o ca $X^{T}X$.
      • Tnh cc h s hi quy v h s t do (Intercept v $x_1 x_{11}$) bng cch nhn: $(X^{T}X)^{-1}X^{T}y$

  • predict(X): D on gi tr ca bin mc tiu (DIQ010) da trn cc feature ca d liu u vo.

    • S dng cc h s hi quy v h s t do tnh ton gi tr d on cho tng mu trong $X$.

  • MSE(y_true, y_pred): Tnh Mean Squared Error (MSE) - sai s bnh phng trung bnh gia gi tr thc v gi tr d on.

  • R2_Adjusted(y_true, y_pred): Tnh $R^2$ Adjusted - h s xc nh (thng s iu chnh ph hp vi m hnh a bin c lp), cho bit t l phng sai ca bin mc tiu c gii thch bi cc bin c lp.

  • forward_selection (X_train, y_train, X_test, y_test): Chn t hp cc feature cho ra model c kt qu tt da trn phng php Foward Selection.

  • Forward Selection: Forward Selection, Backward Elimination l hai phng php ph bin dng gip chn lc bin s trong m hnh hi quy. Trong project ny, Foward Selection c s dng v mt s l do nh sau:

    • Backward Elimination yu cu c s dng p-value, lm tng phc tp ca thut ton cng nh phi trch p-value t mt th vin hoc tnh tch phn xp x phc tp.
    • Forward Selection ph hp cho cc model khng c qu nhiu bin s (feature).
    • Forward Selection bt u vi mt m hnh rng v dn dn thm cc bin, gip d dng theo di qu trnh chn la bin.

  • Quy trnh Forward Selection:

    • Khi to:
      • Bt u vi tp hp cc feature cn li (tt c cc feature ban u).
      • Khi to mt tp hp cc feature chn (ban u rng).
    • La chn feature bng thng s MSE (ng tin hn):
      • i vi mi feature cn li, thm feature vo tp hp cc feature chn v train m hnh hi quy tuyn tnh.
      • nh gi m hnh bng cch tnh Mean Squared Error (MSE) trn tp test.
      • Ghi li MSE cho tng feature khi c thm vo m hnh.
    • Chn feature tt nht:
      • Chn feature m khi c thm vo m hnh, lm gim MSE nhiu nht.
      • Thm feature vo tp hp cc feature chn v loi b n khi tp hp cc feature cn li.
    • Lp li quy trnh cho n khi khng c s ci thin ng k trong MSE khi thm cc feature cn li.

  • Kt qu: Danh sch cc feature c chn s l t hp tt nht cho m hnh hi quy tuyn tnh.

# T xy dng m hnh hi quy tuyn tnh
class LinearRegressionModel:
    def __init__(self):
        self.coef_ = None # H s hi quy
        self.intercept_ = None # H s t do (bias)

    def fit(self, X, y):
        # Thm ct bias (h s t do)
        X = X.copy() # To bn sao ca X  khng nh hng n d liu gc
        X['intercept'] = 1 # Thm ct bias vi gi tr tt c l 1 (h s t do, ch yu  cho x l x0 = 1)
        # Chuyn i dataframe thnh danh sch cc danh sch
        X_matrix = X.values.tolist() # Chuyn dataframe thnh list
        y_matrix = y.values.reshape(-1, 1).tolist() # reshape(-1, 1)  chuyn v dng ct
        
        # Tnh ton cc h s hi quy s dng cng thc: x^ = (X^T*X)^(-1)*X^T*y
        # CC FUNCTION BUILLD-IN S DNG  TNH TON:
        # zip(): Trn cc phn t t cc danh sch hoc b gi tr.
            # V d: zip([1, 2, 3], [4, 5, 6]) -> [(1, 4), (2, 5), (3, 6)] - VD Tham kho https://mathematica.stackexchange.com/questions/89441/what-can-i-use-as-an-equivalent-of-python-zip-in-mathematica-inside-of-sum
            # * trong zip(* ): c s dng  truyn mt danh sch cc i s khng tn cho mt hm.
        # sum(): Tnh tng cc phn t trong danh sch.
        # list(): Chuyn i mt i tng c th lp thnh mt danh sch.
        # map(): nh x mi phn t ca danh sch vo mt hm.
        
        XT = matrix(X_matrix).transpose() # Chuyn v ma trn X
        if (XT is None):
            return None # Nu ma trn khng kh ngh th tr v None (phng h)
        
        # Tnh ton cc h s hi quy
        # B1: Tnh X^T*X
        # B2: Tnh X^T*y
        # B3: Tnh (X^T*X)^(-1)
        # B4: Tnh h s hi quy x^ = (X^T*X)^(-1)*X^T*y
        
        # Hm sum(a * b for a, b in zip(X1, X2))  tnh tng ca tch ca cc phn t trong 2 list (VD: [1, 2, 3] v [4, 5, 6] -> 1*4 + 2*5 + 3*6) thay v dng vng lp
        XTX = [[sum(a * b for a, b in zip(XT_row, X_col)) for X_col in zip(*X_matrix)] for XT_row in XT] # Tham kho cch lm t https://stackoverflow.com/questions/22975532/sum-values-of-array-using-zip
        XTy = [[sum(a * b for a, b in zip(XT_row, y_col)) for y_col in zip(*y_matrix)] for XT_row in XT]
        
        XTX_inv = matrix(XTX).gauss_jordan_inverse()
        x_hat = [[sum(a * b for a, b in zip(XTX_inv_row, XTy_col)) for XTy_col in zip(*XTy)] for XTX_inv_row in XTX_inv] # Tnh h s hi quy x^ = (X^T*X)^(-1)*X^T*y
        # Lu cc h s hi quy
        self.coef_ = [b[0] for b in x_hat[:-1]] # Ly tt c cc h s tr h s t do (h s t do nm  cui)
        self.intercept_ = x_hat[-1][0] # Ly h s t do

    def predict(self, X):
        # D on gi tr da trn cc h s hi quy
        return [sum(x * coef for x, coef in zip(row, self.coef_)) + self.intercept_ for row in X.values.tolist()] # tnh tng ca x*coef + intercept 
    
    def MSE(self, y_true, y_pred):
        # Tnh Mean Squared Error - Sai s bnh phng trung bnh = 1/n * (y_true - y_pred)^2 
        return ((y_true - y_pred) ** 2).mean() # .mean()  tnh gi tr trung bnh
    
    def R2_adjusted(self, y_true, y_pred, X):
        # Tnh R2 adjusted: R^2 = 1 - (1 - R^2) * (n - 1) / (n - p - 1)
        n = len(y_true) # S lng mu
        p = len(self.coef_) # S lng feature
        R2 = 1 - self.MSE(y_true, y_pred) / ((y_true - y_true.mean()) ** 2).mean() # Tnh R^2
        return 1 - (1 - R2) * (n - 1) / (n - p - 1) # Tnh R^2 adjusted

    def forward_selection(self, X_train, y_train, X_test, y_test):
        # Chn ra cc feature tt nht da trn forward selection
        remaining_features = list(X_train.columns)
        selected_features = []
        best_mse = float('inf') # t gi tr MSE ban u l v cc ( trnh trng hp khng tm c feature no tt hn)
        best_model = None

        while remaining_features:
            feature_mse = {}
            for feature in remaining_features:
                current_features = selected_features + [feature] # Th thm feature vo cc feature  chn
                model = LinearRegressionModel()
                model.fit(X_train[current_features], y_train)
                y_pred = model.predict(X_test[current_features]) # D on gi tr
                mse = model.MSE(y_test, y_pred) 
                feature_mse[feature] = mse
                # print(f'Using features {current_features}, MSE = {mse}') # In ra tng bc  kim tra 
            
            best_feature = min(feature_mse, key=feature_mse.get) # Chn ra feature tt nht (c MSE nh nht)
            if feature_mse[best_feature] < best_mse: # Nu feature tt nht tt hn feature tt nht hin ti
                best_mse = feature_mse[best_feature] 
                selected_features.append(best_feature) # Thm feature tt nht vo selected_features
                remaining_features.remove(best_feature) # Loi b feature  chn ra khi remaining_features
                best_model = model
            else: # Nu khng tm c feature no tt hn th dng
                break

        return best_model, selected_features, best_mse

== PHƯƠNG PHÁP ĐÁNH GIÁ MODEL ==¶

Mean Squared Error (MSE):¶

  • ngha:
    • Mean Squared Error (MSE) l gi tr trung bnh ca bnh phng sai s gia gi tr thc t v gi tr d on.
    • Trong bi ton d on loi Tiu ng, thng s ny s tng ng vi lch gia gi tr DIQ010 d on c t cc t m hnh v gi tr thc t trch c t h s hi quy.

Ch s MSE cng nh th m hnh cng tt (sai s cng nh).

  • Cng thc:
$$MSE = \frac{1}{n} \times \sum_{i=1}^n (Y_i - \hat{Y}_i)$$
  • Trong :
    • $n$: S lng mu.
    • $Y_i$: y_true (gi tr thc t ca bin ph thuc).
    • $\hat{Y}_i$: y_pred (gi tr d on ca bin ph thuc)

$R^2$ Adjusted Score (coefficient of determination):¶

Ý nghĩa của $R^2$:¶

  • $R^2$ Score (H s xc nh) l mt ch s thng k cho bit mc ph hp ca m hnh hi quy tuyn tnh. N biu th t l phng sai ca bin mc tiu c gii thch bi cc c trng trong m hnh. $R^2$ khng gim khi thm bin c lp mi.
  • Trong bi ton loi Tiu ng, $R^2$ Score cho bit t l phng sai ca bin mc tiu (DIQ010) c gii thch bi cc bin c lp (7 feature ca sc khe). $R^2$ dao ng t $01$:

$R^2=1$: Tt c phng sai ca bin mc tiu c gii thch bi cc bin c lp (m hnh hon ho).
$R^2=0$: M hnh khng gii thch c bt k phng sai no ca bin mc tiu.
NGOI RA $\rightarrow$ $R^2 < 0$: M hnh c fit mt cch khng hp l.

  • TUY NHIN: Khi thm bin c lp vo m hnh, $R^2$ c xu hng tng, ngay c khi bin khng thc s ci thin m hnh. iu ny dn n mt vn gi l *overfitting* (m hnh tr nn qu phc tp v hot ng km trn d liu mi). Cng chnh v hn ch ny m ta s s dng $R^2$ Adjusted cho d on loi Tiu ng.

Ý nghĩa của $R^2$ Adjusted:¶

  • $R^2$ Adjusted l mt phin bn iu chnh ca $R^2$ tnh n s lng bin c lp (features) trong m hnh hi quy.
  • $R^2$ Adjusted ch tng khi bin mi thc s ci thin m hnh, tc l lm gim sai s d on ca m hnh nhiu hn so vi mc m bn mong i t vic tng thm bin ch do ngu nhin.

Nu thm vo mt bin m khng ci thin hoc thm ch lm gim cht lng m hnh, $R^2$ Adjusted s gim, gip pht hin rng bin khng hu ch. $R^2$ Adjusted c th gim nu bin mi khng thc s gip ch cho m hnh.

Sử dụng $R^2$ Adjusted thay vì $R^2$ trong mô hình hồi quy tuyến tính là hợp lý hơn vì các lý do sau:¶

  1. So Snh Gia Cc M Hnh Khc Nhau:
    • Khi xy dng v so snh nhiu m hnh vi cc s lng c trng khc nhau, $R^2$ Adjusted l mt thc o tt hn so snh v n cung cp mt ci nhn cng bng hn v hiu sut ca cc m hnh.
    • $R^2$ Adjusted cho php nh gi xem liu vic thm cc c trng mi vo m hnh c thc s ci thin kh nng d on hay khng, thay v ch to ra mt m hnh phc tp hn.
  2. Ph Hp Vi M Hnh Hi Quy a c Trng:
    • Trong bi cnh d on loi Tiu ng vi nhiu c trng (features) khc nhau, m hnh c th tr nn phc tp v d dn n overfitting nu khng c cch iu chnh ph hp. $R^2$ Adjusted gip m hnh duy tr s n gin v trnh vic thm vo cc c trng khng cn thit.

  • Cng thc:

    • $R^2$: $$R^2 = 1 - \frac{\sum_{i=1}^n (Y_i - \hat{Y}_i)^2}{\sum_{i=1}^n (Y_i - \={Y}_i)^2} = 1 - \frac{RSS}{TSS} = \frac{ESS}{TSS}$$
    • $R^2$ Adjusted iu chnh li gi tr ca $R^2$ phn nh ng hn gi tr thc ca cc bin c lp c thm vo m hnh: $$R_{adjusted}^2 = 1 - \frac{1 - R^2}{n-K-1} * (n-1)= 1 - \frac{\frac{RSS}{n - K}}{\frac{TSS}{n - 1}}$$

  • Trong :

    • $n$: S lng mu.
    • $K$: S lng bin c lp (s feature) trong m hnh.
    • $Y_i$: y_true (gi tr thc t ca bin ph thuc).
    • $\hat{Y}_i$: y_pred (gi tr d on ca bin ph thuc).
    • $\=Y_i$: y_true.mean() (gi tr trung bnh ca bin ph thuc).

  • Ngoi ra:

    • ESS: explained sum of squares. $$ESS = \sum_{i=1}^n (\hat{Y}_i - \={Y}_i)^2$$
    • RSS: Tng bnh phng ca phn d (Residual Sum of Squares), tc l tng bnh phng cc sai s gia gi tr thc t v gi tr d on. $$RSS = \sum_{i=1}^n (Y_i - \hat{Y}_i)^2$$
    • TSS: Tng bnh phng tng th (Total Sum of Squares), tc l tng bnh phng cc sai s gia gi tr thc t v gi tr trung bnh ca n. $$TSS = RSS + ESS = \sum_{i=1}^n (Y_i - \={Y}_i)^2$$
